Description

Saneful is a game that provides entertainment and satire based on the global pandemic of 2020. As a team, we wanted to build something that would bring some positivity in times that are not so positive. The game requires you to make decisions based on core game logic to survive while also allowing users to save their progress. Additional features include and instructional dashboard and a fully functioning leaderboard.

Technology Used

React.js, CSS3, HTML5, Jest, Node.js, Express.js, REST, Mocha, Chai, PostgreSQL, Heroku, Vercel
